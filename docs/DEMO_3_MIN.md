# WeftEnd 3-Minute Demo

1) Run doctor + demo day:
   - `./scripts/weftend_doctor.ps1`
   - `./scripts/weftend_demo_day.ps1`
2) Open Import Studio: `http://localhost:5173/test/harness/import_studio_v1.html`
3) Paste HTML, click `Blockify`, then `Preview`.
4) Click `Run demo interactions` and see `WARN` with `CAP_DENY` in the report.
5) Click `Export Snapshot`, refresh, then `Load Snapshot` and confirm the digest matches.

Plain English: Proven means the checks passed; scars mean changes or fixes are recorded and visible.
