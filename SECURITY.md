# SECURITY.md

Security policy (v0)

Please report security issues by opening a GitHub issue with the title prefix:
SECURITY: <short summary>

Do not include secrets or exploit code in the initial report.
We will respond in the issue and request details if needed.

No guarantees
- No SLA or response time is guaranteed for v0.
- We will acknowledge reports and triage in good faith.

Scope
- WeftEnd CLI, host, and receipts pipeline.
- Windows shell integration scripts.
